# PyCraft — Build the Web with Python

## Overview
PyCraft is a modern web interface concept that promotes building web applications using Python as a primary language. It presents a clean, editorial-style landing page that highlights Python’s capabilities beyond data science, focusing on full-stack web development.

The project demonstrates how Python frameworks can be used to create responsive, production-ready web applications with minimal reliance on JavaScript.

## Purpose
This project is designed to:
- Showcase Python as a viable tool for modern web development  
- Present different Python web frameworks in a structured interface  
- Provide a visually engaging landing page for educational or promotional use  

## Features
- Fixed navigation with smooth scrolling  
- Responsive hero section with split layout  
- Styled code preview block  
- Feature grid highlighting Python advantages  
- Framework showcase section  
- Statistics section for credibility and context  
- Call-to-action section for user engagement  
- Fully responsive design for different screen sizes
